# Packages & naming

| Name | Role |
|------|------|
| **tyohncn** (repo / CLI bin) | `npx tyohncn` — init / add / apply |
| **@tyohn/registry** | Base UI components (`cn-*`), theme CSS, style packs |
| **@tyohn/studio** | Web catalog + token inspectors + CSS export |
| **@tyohn/docs** | Documentation site |

npm scope: **`@tyohn/*`**. Public CLI package name: **`tyohncn`** (matches the repo).

Package manager: **pnpm** workspaces (Node ≥ 22).
