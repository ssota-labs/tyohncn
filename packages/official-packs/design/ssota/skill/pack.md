# ssota design pack

Brand density pack for [ssota](https://github.com/ssota-labs). Scope class: `style-ssota`.

## Install

```bash
tyohncn pack add ssota
# or
tyohncn apply --style ssota
```

Wrap the app root with `className="style-ssota"`.

## Contents

- **Style** (`style.css`) — `--cn-*` density + chrome for button / input / card
- **Theme** (`theme.css`) — semantic `:root` / `.dark` tokens (`--primary`, `--radius`, …)

Style answers density; theme answers brand color. Components keep `cn-*` hooks.
